# Назва проєкту
**TelaGallery** — платформа для збереження, організації та обміну візуальними ідеями.

## 1. Опис ідеї проєкту
TelaGallery — це вебзастосунок, який дозволяє користувачам зберігати зображення, ілюстрації, дизайни та інші візуальні матеріали у персональні колекції (дошки натхнення). Платформа орієнтована на творчих людей: дизайнерів, студентів, митців та всіх, хто шукає ідеї для навчання чи роботи.

## 2. Мета проєкту
Створити зручний цифровий простір для пошуку натхнення, збереження ідей та їх структурованої організації.

## 3. Основні користувачі
- Студенти
- Дизайнери та митці
- Контент-кріейтори
- Загальні користувачі

## 4. Ключова функціональність
- Реєстрація та авторизація
- Створення галерей (дошок)
- Додавання/видалення зображень
- Пошук по тегам і категоріям
- Збереження інших публікацій

## 5. Функціональні вимоги
- Система повинна забезпечувати реєстрацію та авторизацію користувачів.
- Користувач повинен мати можливість створювати, редагувати та видаляти галереї.
- Користувач повинен мати можливість додавати зображення до галерей.
- Система повинна підтримувати пошук зображень за ключовими словами.
- Система повинна відображати популярний контент.

## 6. Нефункціональні вимоги
- Інтерфейс має бути простим і зрозумілим.
- Час завантаження сторінок не повинен перевищувати 2–3 секунди.
- Дані користувачів мають зберігатися відповідно до вимог безпеки.

## Висновок
Практична робота №1 містить опис ідеї проєкту TelaGallery, мету, цільову аудиторію та базові вимоги. Це основа для подальших робіт.
